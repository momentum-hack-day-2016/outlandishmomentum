Introduction

A Work in progress! This is a framework for a single Voter Identification database with availability tools:


  • A generalised query interface
  • Machine accessible data input
  • The input method needs to update or add depending on whether we know the person or not
  • A framework for value added processes like classifiers to get at the database
  • Tools to add columns
